# src/

- **App.jsx** — Router principal, importa todos os componentes
- **Layout.jsx** — Navigation rail, topbar, FAB
- **main.jsx** — Entry point React
- **index.css** — Todos os estilos (Material Design 3 custom)
- **pages.config.js** — Títulos e config das páginas
- **lib/** — Tokens, dados, DB, nav config
- **api/** — Backend.js (todas as funções do servidor)
- **hooks/** — useToast, useNavigation
- **utils/** — Helpers
- **components/** — Componentes organizados por feature
